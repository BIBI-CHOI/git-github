
현재 디렉터리 살펴보기
    1. 물결표시(~) : 현재 홈 디렉터리에 있다는 의미
    2. pwd : 현재 위치의 경로 출력
    3. ls : 디렉터리와 파일 이름 출력
        -l 상세정보를 함께 표시
        -a 숨긴 파일과 디렉터리 표시
        -la, -al, -r, -t

터미널 창에서 디렉터리 이동하기
    1. cd .. : 상위 디렉터리 이동
    2. cd 이동할 디렉터리 이름 : 하위 및 지정 경로 이동
    3. cd ~ : 홈 디렉터리 이동
    4. cd 디스크드라이브: 해당 디스크드라이브로 이동
        $ cd e:

리눅스에서 디렉터리를 나타내는 기호
    ~  : 사용자 디렉터리(홈 디렉터리)
    ./ : 현재 사용자가 작업 중인 디렉터리
    ../: 현재 디렉터리의 상위 디렉터리

터미널 창에서 디렉터리 만들기 및 삭제하기
    1. mkdir : 디렉터리 만들기
    2. rm : 디렉터리 삭제
        -r 디렉터리 안에 있는 하위 디렉터리와 파일까지 삭제

       파일삭제
       rm file1.txt
       rm *.dat
       rm *
       
       디렉터리 삭제
       rm -r  dir1/
       rm -rf dir1/ : f옵션을 사용하면 경고 없이 모두 강제로 삭제
    
     * 주의:삭제할 디렉터리의 상위 디렉터리에서 rm 명령 실행

vim, vi에서 텍스트 문서 만들기
    1. vim test.txt
    2. '입력모드(i 또는 a)', 'ex모드(ESC)'
    3. '입력모드'에서 입력/수정
    4. 'ex모드'에서 콜론(:) 입력 후
        ':wq'명령 + Enter 저장하고 편집기 끝내기
        ':q!'명령 + Enter 저장하지 않고 편집기 종료

텍스트 문서 내용 확인하기
    1. cat 명령
        cat test.txt
